Tecnologias do Site

HTML5 e CSS3

Construído com uma base sólida de HTML5 semântico, CSS3 e Flexbox, este site combina design moderno e funcionalidade. O layout responsivo, possibilitado pelo Flexbox, garante uma adaptação perfeita a diferentes tamanhos de tela.

Design Responsivo

Com este portfólio, busquei unir um design moderno e funcional com a praticidade de um layout responsivo. Utilizando media queries CSS3, garanti que meus projetos sejam exibidos de forma clara e intuitiva em qualquer dispositivo, proporcionando uma experiência de navegação única.

Ferramentas e Outros

Desenvolvi um menu móvel responsivo utilizando JavaScript puro. Através da manipulação de classes CSS e de eventos de clique, criei uma experiência de navegação intuitiva e adaptada a diferentes dispositivos. Este componente demonstra minha habilidade em criar interfaces.
